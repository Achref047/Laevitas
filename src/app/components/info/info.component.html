This is the initialization page where you enter your Laevitas APi Token.
The API Token is necessary for the other pages to function and once submitted it can only be changed if you Reset the session.

If the token is not valid or it was not entered. Data Fetching and Trading signals inputs pages won't be active.

Data Fetching

On this page you'll select the Metrics you want to fetch.

The select options dropdown will allow you to select multiple metrics.

After submitting the Options to Fetch a new forum will appear.

The forum contains the necessary parameters to select in order to create a query.

These parameters are linked to the selected options.

After selecting the parameters and pressing Fetch Data the request will be submitted and the data will be fetched.

This process is necessary for the Next Pages to function.

After having acquired data, you can press Fetch Again to reset both backtesting and fetched results.

This process is necessary for the Next Pages to function

Trading Signals Input

On this page, you can create your own strategy.

Select the datapoints and technical indicators you wish to use and specify their parameters

For each strategy you can specify the buy and sell signals

IF the entries are all valid : The submission of these parameters will create a new button Show Calculated Data

If the chosen rules were submitted: Backtesting, Used technical indicators Charts and Bot configuration panels will become available

You can control the entry point, position size and capital for the Backtester

You can view the used technical indicators

You can configurate the bot's parameters, from this tab you can reset and try another strategy or confirm

The confirmation is validated when the send button is pressed and the information needed to lunch the trade bot will be submitted

The send button is unlocked only after pressing Confirm and providing Bitfinex Tokens

If sent successfully, a success message will be shown and the data is safely transmitted to lunch a trading bot.

Bitfinex

On the sidebar, you need to fill in your Bitfinex API Credentials : KEY & SECRET then submit

If they are valid, A dropdown will appear on the main page where you can choose which info you want to display

After Submitting, a Table containing the requested info will appear.
